Dungeon Core Asset Pack

This folder is the bundled default asset pack for Wayfinder Atelier.

- Place curated room/corridor/door textures here.
- Keep filenames stable when possible for deterministic map skin rendering.
- Recommended source sets:
  - ambientCG (CC0) for stone/parchment material textures
  - Kenney dungeon packs (CC0) for pixel style textures

The app currently ships with procedural fallback patterns if no image assets are present.

Included starter textures:

- `stonekeep-room.svg`
- `stonekeep-corridor.svg`
- `parchment-room.svg`
- `parchment-corridor.svg`
- `pixel-room.svg`
- `pixel-corridor.svg`
