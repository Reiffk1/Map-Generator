# Kenney CC0 Tileset Assets

Place the following CC0-licensed tileset PNG atlases in this directory:

## Required Files

1. **tiny-dungeon.png** — From [Kenney Tiny Dungeon](https://kenney.nl/assets/tiny-dungeon)
   - 16x16 pixel tile size
   - Download from kenney.nl and place the tilemap atlas here

2. **scribble-dungeons.png** — From [Kenney Scribble Dungeons](https://kenney.nl/assets/scribble-dungeons)  
   - 16x16 pixel tile size
   - Download from kenney.nl and place the tilemap atlas here

## License

Both packs are CC0 (public domain) by Kenney.nl.
No attribution required, but credit is appreciated.

## How to Install

1. Download from https://kenney.nl/assets/tiny-dungeon
2. Extract and copy the tilemap PNG to this folder as `tiny-dungeon.png`
3. Download from https://kenney.nl/assets/scribble-dungeons
4. Extract and copy the tilemap PNG to this folder as `scribble-dungeons.png`
5. Select the pack in the Inspector > Map > Asset Pack dropdown
